Celeri v1.1.0

latin - adjective: fast, swift, rapid

A fast and simple HTTP server library for Node.js

Getting Started

Getting started is easy! First, install Celeri: 


				$ npm install --save @celeri/http-server

Then, load in the module and make a server! 


				const { createServer } = require('@celeri/http-server');

				const app = createServer();
				app.use(app.router);

				app.get('/hello-world')
					.use(({ req, res }) => {
						res.statusCode = 200;
						res.setHeader('Content-Type', 'text/plain');
						res.end('Hello, World!');
					});

				app.server.listen(8080);

Take a look at the example project on our github for an example of a fully functional server or the reference documentation for more detailed information.

Modules

Celeri has many supporting modules that you can choose to use to add extra features to your server. Here are some of the most common modules:

@celeri/router
The router that Celeri uses to route incoming requests to the right endpoint. There are two implementations available depending on your needs - RegexRouter and TrieRouter.
@celeri/body-parser
Automatically parses incoming request bodies and makes them available to your endpoints.
@celeri/query-parser
Automatically parses query strings on incoming requests and makes them available to your endpoints.
@celeri/http-error
Provides a simple interface for throwing errors to be handled as HTTP error responses.
@celeri/express-middleware
Converts middleware written for Express style apps into middlewares that will work with Celeri.
