Installation


						$ npm install --save @celeri/body-parser

bodyParser()


						const { bodyParser } = require('@celeri/body-parser');

						const bodyParserMiddleware = bodyParser();

The body parser middleware parses JSON request bodies of incoming requests, providing the information in the req.body property. Once a middleware is created, it can be used in routes, or (less efficiently) globally in the app. 


						const bodyParserMiddleware = bodyParser();

						// This comes at a small performance cost
						app.use(bodyParserMiddleware);

						// This is more efficient
						app.use(app.router({ /* ... */ }));
						app.post('/foo')
							.use(bodyParserMiddleware)
							.use(({ req, res }) => { /* ... */ });

Return Value

Returns a body parser middleware function.
